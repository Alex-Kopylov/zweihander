# Codex Harness

Use available file-reading and search tools instead of Claude Code tool names:
`sed` or `nl` through `exec_command` for focused file reads, and `rg` or
`rg --files` through `exec_command` for text and file discovery.

If delegated exploration is useful and the current tool list supports it,
discover and use the Codex multi-agent capability; otherwise perform the audit
in the current context.
