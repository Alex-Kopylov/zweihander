# mermaid-diagrams

Generate and validate Mermaid diagrams with synced Mermaid syntax references.

## Skills

| Skill | Description |
|---|---|
| `mermaid` | Generate Mermaid diagrams from user requirements, using local syntax references for supported diagram types. |
| `mermaid-lint` | Validate Mermaid code blocks by rendering them with `mmdc` and reporting lint status and errors. |

## Notes

- `mermaid` runs `mermaid-lint`, fixes Mermaid syntax errors when possible, and relints before presenting generated diagrams as ready.
- `mermaid-lint` requires the `mmdc` command from `@mermaid-js/mermaid-cli`.
- The synced references are bundled with the plugin so diagram generation can use local syntax documentation.

## Documentation Sync

This plugin includes a GitHub Action that checks the official [mermaid-js/mermaid](https://github.com/mermaid-js/mermaid) repository weekly. It syncs syntax and selected config docs, removes upstream autogenerated warning banners from references, derives diagram metadata from Mermaid docs, and regenerates `skills/mermaid/SKILL.md` plus this README from the synced files.
